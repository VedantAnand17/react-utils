# Hooks

```jsx
import { hooks } from '@dr.pogodin/react-utils';

const {
  useAsyncData,
  useGlobalState,
} = hooks;
```

`useAsyncData(..)` and `useGlobalState(..)` are aliases of the same named hooks
from [`@dr.pogodin/react-global-state`](https://www.npmjs.com/package/@dr.pogodin/react-global-state).
